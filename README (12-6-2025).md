# Delta Software Team -- Social Media AI Integration

This project is a desktop application that brings together three
AI-powered utilities designed to enhance and moderate social media
content:

1.  **AI-Generated Post Summarizer** -- summarizes user posts into
    short, personality-revealing paragraphs.
2.  **AI Subtitle Generator** -- creates `.srt` closed captions from any
    uploaded video.
3.  **Harmful Content Scanner** -- checks both text and media uploads
    for gore, nudity, sexual content, and violent threats, and returns
    accept/reject decisions.

The application is written in **Python 3**, built with **Tkinter**, and
integrates **Google Gemini**, **Whisper**, **Vosk**, and **FFmpeg**.

------------------------------------------------------------------------

## Features

### 1. AI Post Summarizer

-   Users are stored in a `User` class containing:

    -   Username
    -   Up to five posts
    -   Views, likes, comments

-   Engagement weight formula:

        weight = views + 3 * likes + 10 * comments

-   Top three users (by weight) are passed into a single Gemini prompt.

-   Gemini returns a paragraph-style summary for each user, prefixed
    with the username in brackets:

        [username] Summary...

-   Shown live in the right-hand panel of the GUI.

------------------------------------------------------------------------

### 2. AI Subtitle Generator

-   User selects a local video file.
-   `generate_srt_from_video()` handles the full pipeline:
    1.  Extract audio using FFmpeg at 16kHz, mono.
    2.  Run ASR via:
        -   Whisper (`transcribe_with_whisper_local`)
        -   OR Vosk (`transcribe_with_vosk_streaming`)
    3.  Clean raw transcription using Gemini:
        -   Capitalization
        -   Punctuation
        -   Minor smoothing while keeping wording accurate
    4.  Merge and convert segments into `.srt` format.
-   Output file works in VLC and other media players.

## Architecture

### High-Level Structure

-   **GUI Layer (Tkinter)** 
    Includes three tabs (Summarizer, Subtitler, Scanner) and a
    right-hand output window using a bound `StringVar`.

-   **Domain Layer** 
    Contains the `User` class and helpers for data management:

    -   Sorting users
    -   Updating posts
    -   Loading user data into the GUI

-   **Service Layer** 
    Handles integrations:

    -   Gemini for NLP, caption cleanup, and moderation 
    -   Whisper/Vosk for ASR 
    -   FFmpeg for audio extraction 
    -   Subtitle generator pipeline (in `captioner.py`)

### Architectural Style

-   Primarily **Layered Architecture**
-   Subtitle generator uses a **Pipeline** pattern
-   ASR backend selection uses a light **Strategy Pattern**

## Setup & Requirements

1.  Install Python 3.11.x 

2.  Install required packages:

    -   `google-genai` 
    -   `faster-whisper` or `whisper` or `openai-whisper`
    -   `vosk` (optional) 
    -   `pydub` 
    -   `ffmpeg` (must be installed and in PATH) 
	-   `tkinter`(if not already installed)

3.  Set environment variable:

        GEMINI_API_KEY = constantly changing when new key needed

4.  Place ai_gui.py and ai_captioner.py into a compiler project.

4.  Run the main GUI script via the compiler.

## Usage Guide

### Summarizer Tab

-   Add users and enter posts, views, likes, and comments 
-   Click **Update Posts** 
-   Click **Generate AI Summary**

### Subtitler Tab

-   Browse and select a video 
-   Click **Scan File** 
-   App outputs a generated `.srt` file path

### Scanner Tab

-   For **text**: type and click **Text Check** 
-   For **files**: browse and click **Scan File** 
-   Output indicates *accepted* or *rejected* content
